# 🎓 Cadastro de alunos

Um projeto simples feito com HTML, css e javascript para praticar conceitos basícos de programação e manipulação do DOM 

# 🚀 Funcionalidades: 

- Adicionar alunos ✅
- Mostrar alunos na tela ✅
- filtrar aprovados ✅
- Alterar cor baseado na nota ✅

# 🎯 Objetivo do projeto

Esse projeto foi criado para praticar:


-  Variáveis
- Arrays
- Objetos
- Funções
- Arrow Functions
- Condicionais (if)
- Eventos
- DOM
 -map()
- filter()

# 🤔 Como funciona:

1- Digite o nome do aluno
2- Digite a nota
3- Clique em adicionar
4- O aluno aparece na lista

se a nota for: 
-maior ou igual a 7 = ✅
-menor que 7 = ❌
